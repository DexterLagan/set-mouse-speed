# set-mouse-speed
A tiny, very fast native Windows program that sets the mouse cursor sensitivity from the command line.
Syntax:
<pre>
set-mouse-speed [1-10]
</pre>
Example:
<pre>
set-mouse-speed 5
</pre>
How to compile:
<pre>
newlisp -x set-mouse-speed.lsp
</pre>
Sets the mouse cursor sensitivity to 5 out of 10.
